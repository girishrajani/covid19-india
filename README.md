# covid19-india-count
<h2>This projects shows count of COVID-19 cases around India and its States and Union Territories. </h2>
<h3>This project scraps data from https://www.mygov.in/covid-19</h3>
<h1>Dependencies:</h1> 

```
pip install requests
pip install Pillow
pip install bs4
```
